    I. First Page

    When a user accesses this aplication he has 2 choices: if he has an account he will introduce his username and his password or, if he doesn't have an account, he will complete a form in order to create an account.

    II. Map Page

    After the user is logged/registered he can choose the country he is interested in.

    III. Choices Page

    When he chooses a country he will automatically be redirected to a page where he will be able to choose the kind of person he is looking gifts for(he will enter the age, gender and also the month when he will travel). By knowing these characteristics our application can provide more precise recommandations.

    IV. Results Page

    On this page the user will be able to see the recommended souvenirs with a minimal description and an aproximated price. If he likes one of this souvenirs he can add it on his profile page by pressing the button "I'd like that". In addition the user can download this recommandations in different formats(HTML, JSON, CSV, XML).

    V. Profile Page

    On the profile page the user will see the souvenirs he selected grouped according to their countries. Also the informations he/she entered when the account was created will also appear an this page.
