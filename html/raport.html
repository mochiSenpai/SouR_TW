  • I. About us

    This application will help its users find the perfect souvenirs for their loved ones. It is designed for people who often travel and don't know what gifts to choose. To make the results more accurate the user will enter some informations like the country where he wants souvenirs or for what kind of person these will be(age, gender). The results will be avaible for downloading in other formats like HTML, JSON, XML, CSV.

  • II. Why?

    Everybody who travels a lot knows that it is pretty hard to choose souvenirs, espeacially when you don't know a lot of things about that country's tradition and history. This application comes to help those who want to find the perfect gifts for their dear ones.

  • III.Progress

    In the first phase we were preocupied with how the user will use this application and how it will look. We tried to make it verry easy to use and attractive. After this part was complete we created a database to store the informations about users and souvenirs. The back-end part was implemented using PHP and made the connection between the database and the front-end.
